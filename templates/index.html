    Rubric CSV format

    Five columns, same for every assignment — only the rows change:

    criterion,points,check_type,target,params

    • criterion — display name; becomes a column in the grades CSV
    • points — awarded all-or-nothing when the check passes
    • check_type — one of the checks below
    • target — file glob relative to each student folder (e.g. *.py, index.html)
    • params — key=value pairs separated by ;
